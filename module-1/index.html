Assignment-1: Form Tag

In Day-1 class ,we learn about HTML and its different types of tags. In this assignment form tag is used to create a student information The form tag in HTML is a fundamental element used to create interactive forms on web pages. It serves as a container for various input elements such as text fields, checkboxes, radio buttons, and buttons, allowing users to input data and interact with the webpage The

tag is used to define the structure and behavior of the form, including attributes like action (specifying the URL where the form data is submitted), method (specifying the HTTP method used to send the form data), and enctype (specifying how the form data should be encoded before it is sent to the server). Through the form tag, developers can create seamless user experiences, collect user input, and process data efficiently on the web.
